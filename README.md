### Swiperjs
Chèn thư viện cdn vào WP hoặc HTML

Một số tính năng nổi bật riêng biệt:
- Tạo slider với 1 số item hiển thị 1/2, 1/3...
- Tạo dots dạng thanh trượt

Một số dự án đã dùng: https://www.fikkerts.com/
